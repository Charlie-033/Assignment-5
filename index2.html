Question-1: What are the different ways to select an element in the DOM?

There are some different ways to select an element in the DOM.
1.getElementById(select element by it's id name),
2.getElementsByClassName(select multiple element by their class name),
3.getElementsByTagName(select elements by their tag names like div, p, h2),
4.querySelector(select first element that match the given css selector),
5.querySelectorAll(select all elements that match the given css selector)

Question-2: What is the difference between innerHTML, innerText, and textContent?

1.innerHTML-Sets the HTML content inside an element.
2.innerText-Sets the visible text inside an element.
3.innerContent-Sets the full text content, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation in JavaScript is a pattern that efficiently handles events. Event Delegation is a method in Js where adding a single event listener to a parent element instead of adding event listeners to child elements. The event bubbles up from the child elements to the parent.The parent handle events for all child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling in Js is a method where an event triggered on a child element propagates towards through it's ancestors in DOM. It allows parent elements to response to events triggered by their child element.
